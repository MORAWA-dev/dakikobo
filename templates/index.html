Sources & limites

Base de preuves

Les réponses générales s'appuient sur les documents agricoles locaux indexés dans DakiKobo. Les cartes affichent les sources utilisées quand elles sont disponibles.

Outils déterministes

Les doses d'engrais viennent de l'outil interne, pas du modèle de langage. La météo utilise Open-Meteo et le sol utilise SoilGrids comme signaux indicatifs.

Limites terrain

Les photos, le sol et la météo ne remplacent pas une observation de parcelle. Les maladies, doses exactes et décisions coûteuses doivent être confirmées avec un agent agricole ou une analyse de sol.

DakiKobo privilégie une réponse prudente ou un refus honnête plutôt qu'une précision inventée.
